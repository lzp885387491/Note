# http协议

``` js
# http 协议的数据结构

http协议分为两部分
// http 就相当于一个人  人呢就有 头和身体
1、一个叫请求头
2、一个叫请求体

所以 不管是 服务端 和 浏览器 进行交互，
还是 浏览器发出请求 和 服务端去交互 
# 都必须 发送这样的格式

// 在浏览器中 访问 /home 时，他需要吧 前端访问的 url 地址 发送给服务端
// request ： 请求
const request = {
    headers:{
        // 请求头
        url:'/home'
    },
    body{
        // 请求体
    }
}

```
